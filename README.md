## Dev setup
### For Nix dev environments
```bash
$ nix develop
$ cargo run
```
### For Windows dev environments
Install rust via - https://rustup.rs/
```
// Choose the latest beta or nightly channel
$ rustup toolchain install nightly
$ cargo run
```
